\subsubsection{Detailed Description}
Seven-\/segment display library for A\+V\+R-\/\+G\+CC. 


\begin{DoxyCode}
\textcolor{preprocessor}{#include "\hyperlink{a00005}{segment.h}"} 
\end{DoxyCode}


The library contains functions for controlling the seven-\/segment display (S\+SD) using two shift registers 74\+H\+C595.
